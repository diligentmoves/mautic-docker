# 🚀 Mautic 5 Docker Stack

This is a production-ready Docker Compose setup for Mautic 5, inspired by AutoMautic.

## ✅ Features
- Mautic 5 with Apache
- MariaDB 10.6
- Traefik reverse proxy with HTTPS (Let's Encrypt)
- Persistent volumes
